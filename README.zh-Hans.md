## 新手入门

<!-- 在这里写下项目的详细说明，告诉用户如何使用你的项目。 -->

安装依赖：

```shell
npm install
```

本地开发预览模式：

```shell

npm run dev
```

构建前端资源：

```shell

npm run build
```

以本地服务方式预览前端资源：

```shell

npm run preview
```
